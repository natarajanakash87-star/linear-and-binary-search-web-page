# linear-and-binary-search-web-page
# 🔍 Linear & Binary Search Visualizer

An interactive web application that helps students learn and understand **Linear Search** and **Binary Search** algorithms through live demonstrations and C++ code examples.

## 🚀 Features

- ✅ Interactive **Linear Search Simulator**
- ✅ Interactive **Binary Search Simulator**
- ✅ User-friendly and responsive interface
- ✅ Displays **C++ implementation** of both algorithms
- ✅ Comparison table between Linear Search and Binary Search
- ✅ Instant search result visualization
- ✅ Built using only **HTML, CSS, and JavaScript**

---

## 📸 Preview

Users can:

1. Enter an array of numbers.
2. Enter the target element.
3. Click **Search**.
4. Instantly see whether the element is found and at which index.

Example:

```
Array: 10,20,30,40,50
Target: 30

Result:
Element Found at Index 2
```

---

## 🧠 Algorithms Included

### Linear Search

- Searches elements one by one.
- Works on sorted and unsorted arrays.

**Time Complexity**

| Case | Complexity |
|------|------------|
| Best | O(1) |
| Average | O(n) |
| Worst | O(n) |

---

### Binary Search

- Searches by dividing the search space into halves.
- Requires a sorted array.

**Time Complexity**

| Case | Complexity |
|------|------------|
| Best | O(1) |
| Average | O(log n) |
| Worst | O(log n) |
